# Lezzet Durağı Restaurant Website

A responsive restaurant website developed using HTML, CSS, and Bootstrap.

## Features

* Modern and responsive user interface
* Restaurant home page
* Food menu page
* Reservation page
* User login page
* Customer feedback page
* Food gallery with images
* Mobile-friendly design

## Pages

### Home Page

Provides an introduction to the restaurant and featured content.

### Menu Page

Displays food and beverage options with images.

### Reservation Page

Allows customers to submit reservation requests.

### Login Page

Provides a simple user login interface.

### Feedback Page

Enables customers to leave comments and feedback.

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
